CSS

FLEX YAPISI

Flex kelime anlamı olarak esnek demektir.

Css özelliğinin adı değil yapının adıdır.Bundan dolayı hem kapsayıcı hemde içindeki öğeler için birden fazla flex özeelliği bulunmaktadır.

Flex kullanmadaki amacımız esneklik verdiğimiz yapıları kendi istediğimiz şekilde yön verebilmemizi sağlar istediğimizde sağ sol istediğimizde kolon yada sütun olarak yerlerin belirlemek için kullanırız.

Esneklik yapıcı vereceğimiz birden fazla kutu vb. için en başta onları bir container içine alıyoruz ki container içine attığımız her bir item için de flex yapısı kullanıp onlarıda istediğimiz yer için konumlandıralım

Örnek1 İlk olarak bir çerçeve(container) içine birden fazla kutuyu yan yana koymak istediğimiz zaman contaner a display: flex; yapısını vermemiz gerekiyor

Buradaki containera verilen flex yapısı içine konulan herhangi bir item ı yan yana dizilmesini sağlar.

flex-direction

flex-direction:row ;

flex-direction

flex-direction:row

Varsayılan olarak içindeki nesneleri soldan sağa yan sana sıralar

flex-direction:row-reverse

içindeki nesneleri sağdan sola yan sana sıralar

flex-direction:column

içindeki nesneleri yukarıdan aşağıya alt alta sıralar

flex-direction:column-reverse

içindeki nesneleri aşağıdan yukarıya alt alta sıralar

flex-wrap

flex-wrap:nowrap

varsayılan olarak tüm öğeleri tek satırda yan yana getirir.

flex-wrap:wrap

içindeki nesneleri gerektiğinde birden fazla satırda yukarıdan aşağı doğru sıralar

flex-wrap:wrap-reverse

içindeki nesneleri gerektiğinde birden fazla satırda aşağıdan yukarıya doğru sıralar

flex-flow

Bu özellik flex-direction ve flex-wrap özelliklerinin kısa yoludur. Yani colon olarak içindeki nesneleri yan yana dizilmesini sağlar

justify-content

justify-content:flex-start

Varsayılan olarak içindeki nesneleri banlangçtan sola dayalı hizalar

justify-content:flex-end

içindekileri bittiği yerden sağa dayalı olarak hizalar

justify-content:center

içindeki nesneleri ortalar

justify-content:space-between

içindeki ilk nesne sola dayalı , son nesne sağa dayalı diğerleri ortalı olaara hizalar

justify-content:space-around

ilk ve son nesne hariç diğerlerinin araları eşit olacak şekilde hizalar

justify-content:space-evenly

içinde olan tüm nesnelerin arası aynı eşitlikte olaracak şekilde hizalar

align-items

Nesneleri dikey eksende hizalamak için kullanılır.

align-items:flex-start

İçindeki nesneleri yukarıya doğru hizalar

align-items:flex-end

İçindeki nesneleri aşağıya doğru hizalar

align-items:center

içindeki nesneleri ortaya hizalar

align-items:baseline

Nesnelerin içindeki ilk satırı baz alarak ortalama yapar.

align-items:stretch

Varsayılan olarak içindeki nesneleri hizalamaz nesnelerin yüksekliklerini containera göre uzatır.

align-content

Çok satırlı yapıda nesnelerin eksende hizasını ayarlamak için kullanılır.

align-content:flex-start

Nesneleri dikey eksende başta hizalar.

align-content:flex-end

Nesneleri dikey eksende sonda hizalar.

align-content:center

Nesneleri dikey eksende ortada hizalar.

align-content:stretch

Varsayılan olarak nesneleri hizalamaz yüksekliklerini uzatır.

align-content:space-between

İlk satırdaki nesneler başta son satırdaki nesneler sonda olmak üzere satırdaki nesneleri dikey eksende ortalar.

align-content:space-evenly

Nesneleri satırlar arası eşit oranda olacak şekilde dikey eksende ortalar.

Items için Flex özellikleri

Diğer anlatığımız özellikler containera aitti.Şimdi ise nesnelerin içindeki item lara ait özellikleri anlatacağım

.item5{ order : 2 ; } .item2{ order : 5 ; }

Mesela diyelim ki container ın içinde 5 kutumuz var item5 yerine 2 yazar item2 yerine 5 yazarak sırasını değiştirebiliriz.Yani öğelerin sıralınışını değiştirmek için order kullanılır.

flex-grow

.item1 { flex-grow: 1; } .item2 { flex-grow: 1; } .item5 { flex-grow: 2; }

Bir item ın diğerine göre daha büyük yapmak için kullanırız.Örnek olarak 3 tane item i olan bir grupta iki iteme 1 değerini verip diğerine 2 değerini verirsek 2 değerine sahip olan item daha büyük olacaktır.

flex-basis

.item3 { flex-basis: 40%; } .item4 { flex-basis: 0; }

Burada 3. ye %40 verip 4. ye 0 verirsek 3. olan kutunun genişliğini değiştirmiş oluruz.

align-self

align-content ile aynı özelliğe sahiptir.Dikey eksende öğelerin nasıl hizalanacağı belirlenir. Tek farkı kapsayıcı yerine bunu öğelere uygulamasıdır.
